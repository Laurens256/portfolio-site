Over mij

Hoi! Ik ben Laurens. Ik ben 18 jaar een woon in Amersfoort. Ik studeer Communication and Multimedia Design aan de Hogeschool van Amsterdam, momenteel zit ik in mijn tweede jaar.


Mijn interesse licht voornamelijk bij front-end devolpment. De talen die ik hierbij toepas zijn; HTML, CSS en Javascript. Ik ben nu al zo'n 2 jaar intensief bezig met deze talen. Naast school, doe ik voor mezelf ook projecten, vooral gericht op Javascript.

Innersloth website

Dit project was een opdracht voor het vak; Frontend Development. Voor dit project heb ik de site van Innersloth, een game studio, gedeeltelijk nagemaakt. De focus van het project lag veel op toegankelijkheid, daarom is de site bijvoorbeeld ook te gebruiken door keyboard-only users of mensen die een screen reader gebruiken. Voor dit project kreeg ik een 10.

Ganzenbord

Dit is het eerste, grote zelfstandige project dat ik met Javascript heb gemaakt. Het gaat om een volledig werkende versie van het oud-Hollandse ganzenbord. Deze versie kan je met 2-4 personen spelen en je kan zelfs tegen de computer spelen. Hoewel dit een zelfstandige was, ben ik er wel voor beoordeeld. Ik kreeg voor dit project een 9.

Tamagotchi

Dit project hoort bij het vak; Inleiding Programmeren. Dit was dus ook mijn eerste project waar ik gebruik maakte van Javascript. Het project is een simpele tamagotchi met geldsysteem. Je kan de tamagotchi voeren, aaien en ermee spelen. Met de munten die je verdiend door ermee te spelen, koop je eten en nieuwe outfits. Voor dit project kreeg ik een 9,5.
